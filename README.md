Simple Blog Website for practice.

A Python Flask Website that displays the cards with various stories and informations.

Features:

Main Pages with big header and menu for login, register, about and contact.

User-friendly for reading and commenting.

Can be accessed with your account after registration, users that are logged in can comment on the stories, every user can message with email form on the contact page.

Usage:

Clone the repository:

git clone https://github.com/MarkoZivkovic1984/Blog_Website.git

Install dependencies:

pip install -r requirements.txt
